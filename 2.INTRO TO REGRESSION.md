# REGRESSION :
- A statistical method that help us understanding and predict the relation between variables 
- Describes how one variable changes ( dependent ) based on the another variable ( independent variable ) 
- **Dependent variable** : we r trying to predict or explain
- **Independent variable** : used to predict changes in dependent variable 
- Ex : 
  1. predicting salary based on years of experience 
	2. Predicting exam score based on study hours 
	3. Predicting resale price based on vehicle age 
## Types :
- **Linear regression** : 2 variables ( 1 dependent & 1 independent ) – linear graph
- **Multi linear regression** : multiple variables ( 1 dependent and more than 1 independent ) – linear graph
- **Polynomial regression** : polynomial graph
